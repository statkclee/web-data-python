웹에 있는 데이터 작업

이번 수업은 웹으로부터 데이터를 소비하는 방법과 다른 사람이 사용할 데이터를 생성하는 방법을 설명한다.

전제조건

이번 수업은 기본적인 파이썬 프로그래밍 이해를 학습자가 갖고 있는 것을 가정한다. 그리고 다음을 수행할 수 있어야 된다:

  • 파이썬 라이브러리 설치
  • 문자열과 리스트 사용
  • 간단한 함수 작성과 함수 호출
  • 수업 중에 인터넷에 접속

사전 준비사항

파이썬과 네트워크 접속이 되어있어야 된다. 컴퓨터에 파이썬 라이브러리를 설치할 필요가 있다.

학습 주제

  1. 데이터 입수
  2. CSV 데이터 처리
  3. 오류(Error) 처리와 일반화
  4. 시각화
  5. 데이터 게시
  6. 데이터를 찾울 수 있게 만들기
